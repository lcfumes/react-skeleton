## Project Contents ##


###First Step###

```
npm install
```

### DEV Run ###

```
npm run dev
```

### DEV Deploy (It will be create a package and run with Docker) ###

```
npm run deploy
```

### Test ###

```
npm run test
```

### Generate Coverage ###

```
npm run coverage
```


### Generate Coverage ###

```
docker-compose up -d
```

You must add in yours /etc/hosts

```
127.0.0.1 webproject.dev
```

Browser access

```
http://webproject.dev
```
